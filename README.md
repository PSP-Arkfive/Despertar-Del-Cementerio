# Despertar Del Cementerio
Legendary unbricking tool for the Sony PSP.


## Installation

- Copy `DC10` to `/PSP/GAME/` on the Memory Stick.
- Download the latest 6.61 firmware, rename it to `661.PBP` or `661GO.PBP` (for PSP Go) and copy to your Memory Stick.
- Run the app.
- Use a recovery method such as `Pandora Battery`, `Baryon Sweeper` or `New cIPL`.


## Features

- Works on all retail PSP models (1K, 2K, 3K, Go and Street).
- Allows doing a fresh installation of `6.61` firmware with or without `ARK`.
- Can boot `6.61 ARK` from Memory Stick without doing any permanent changes to the device.
- Backup and restore nand.
- Backup and restore IDStorage
- Permanently change UMD drive region.
- and more...


## History

- Version 1: supported only PSP-1000 models and installs 3.71 M33-2 custom firmware or Sony 3.71 official firmware.
- Version 2: fixed a bug where some PSP-1000 models would brick using Despertar del Cementerio v1.
- Version 3: added kernel support for some PSP-2000 Slim models allowing the service menu to be displayed.
- Version 4: installs 3.80 M33-5 custom firmware or Sony 3.80 official firmware.
- Version 5: installs 3.90 M33-3 custom firmware or Sony 3.90 official firmware.
- Version 6: added support for TA-088v2 motherboards, added Test M33 feature to boot 4.01M33-2 from memory stick, and installs 4.01 M33-2 custom firmware or Sony 4.01 official firmware.
- Version 7: adding the ability to regenerate some vital parts of the ID Storage and installs 4.01 M33-2 custom firmware or Sony 4.01 official firmware.
- Version 8: installs 5.00 M33-4 custom firmware or Sony 5.00 official firmware.
- Version 9: was reverse engineered from v7 and v8 by Balika001, added support for PSP-2000 TA-088v3 motherboards, added support for all first generation PSP-3000 models (03g), and installs 5.00 M33-7 custom firmware or Sony 5.00 official firmware.
- Version 9.01: also by Balika001, added Testing Tool M33 feature, and installs 5.02 M33 custom firmware or Sony 5.02 official firmware.
- Version 10: supports flashing a truly permanent and full variant of the 6.61 ARK4 CFW on any PSP model, GO and Street included.
- `Version 10.10`: adds support for idstorage emulation on memory stick, installs `ARK-5`, several other fixes and improvements.
