# Despertar Del Cementerio
Legendary unbricking tool for the Sony PSP.


## Installation

- Copy `DC10` to `/PSP/GAME/` on the Memory Stick.
- Download the latest 6.61 firmware, rename it to `661.PBP` or `661GO.PBP` (for PSP Go) and copy to your Memory Stick.
- Run the app.
- Use a recovery method such as `Pandora Battery`, `Baryon Sweeper` or `New cIPL`.

## Features

- Allows doing a `6.61 ARK` or `6.61 OFW` fresh installations.
- Can boot `6.61 ARK` from Memory Stick without doing any permanent changes.
- Backup and restore nand.
- Change UMD drive region.
- and more...
